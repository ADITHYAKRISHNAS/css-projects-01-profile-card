# Profile Card

A responsive profile card built with HTML and CSS.

## Figma Design

[Profile Card - Community](https://www.figma.com/proto/1zdyvMwes2B8CN4CyQwfNN/profile-card--Community-)

## What I learned

- CSS reset and why it matters
- Flexbox for centering and layout
- `border-radius: 50%` for circular avatars
- `overflow: hidden` for clipping images inside rounded cards
- `object-fit: cover` for images without distortion
- Negative margin for overlapping elements
- `box-shadow` for card depth
- `:hover` pseudo-class for button interactions
- BEM naming convention for CSS classes
- Relative file paths for images

- CSS3
